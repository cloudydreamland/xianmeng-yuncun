## 项目定位

一座藏在云墙后的个人数字村落，用来发布学习笔记、开源项目经历与个人介绍。页面以山窗书屋、百工阁和三山观组成内容入口，让作品集不只是简历条目的重复。

## 技术实现

- 使用 Astro 内容集合与 MDX 管理笔记、项目和元数据，并在构建阶段生成静态页面。
- 通过少量 React Islands 实现云幕入场、全境地图、卷轴侧栏与四时环境切换，控制客户端脚本体积。
- 集成 Pagefind 中文全文搜索，同时提供 RSS、Sitemap、自定义 404 与响应式导航。
- 为减少动态效果偏好、无 JavaScript 场景和键盘导航提供降级支持。

## 当前状态

站点仍在持续维护，后续会继续补充真实笔记、开源项目档案与可验证的个人经历。
