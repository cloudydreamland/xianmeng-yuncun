## 黎明从一页纸开始

山窗还未全亮，风先越过竹林，吹动桌边散开的几页纸。昨夜没有写完的句子仍停在原处，墨迹已经干了，心绪却在晨光中重新有了流向。

生活中总有许多无法立刻解决的事。与其把它们一并搬进梦里，不如在每天清晨留下一点空白：煮一壶茶，整理桌面，然后只完成一件真正重要的事。

> 安宁不是远离一切声音，而是在声音之中，仍能听见自己的呼吸。

## 给日常留一条小路

我开始尝试把复杂的目标拆成很小的路径：读十页书，写三百字，修好一个搁置已久的问题。小路并不壮阔，但它确实通向山顶。

云村也将这样慢慢生长。今天是一间书屋，明天可能多一座桥；重要的不是一次建成，而是每一次回来，都还愿意添上一块石头。

```ts
const today = {
  read: true,
  write: true,
  hurry: false,
};
```

## 窗外

太阳越过峰峦，水面开始发亮。新的一天并没有回答所有问题，但光已经照进来了。
