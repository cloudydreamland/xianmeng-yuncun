# 首页四时地图 V3 人工验收清单

这份清单用于替换或重新生成 `world-detailed-v3-8k.webp` 后做人工检查。首页地图是交互坐标图，构图一致性优先于创意重绘。

## 资产路径

- 母图：`media-originals/world/{dawn,day,dusk,night}/world-detailed-v3-8k.webp`
- 网页衍生图：`public/images/world/{dawn,day,dusk,night}/world-detailed-v3-{1440,1920,2560,3840,5120}.avif/webp`
- 回滚资产：保留 `world-detailed-v2-*`，不要删除。

## 对比检查

- 与 v2 对齐：岛屿轮廓、道路、桥、主要建筑位置不漂移。
- 七境标牌仍落在正确地点：云村、雨桥、星渊、风谷、月潭、雪崖、浮屿·灯巷。
- 星渊光路不能被增强成断裂道路。
- 雪崖建筑、冰瀑和山体不能出现重影或错位。
- 月潭水面可以更细，但不要出现文字伪影或异常月亮。
- 雨桥桥面不能断裂，桥孔和水纹要自然。
- 夜晚图不应过度锐化到星点、云边和灯火出现明显噪点。

## 页面检查

- 大屏高 DPI 下实际候选应能选择 3840 或 5120。
- 移动端仍应加载 960 或 1440 附近资源。
- 切换四个时间段时热点、抽屉、无脚本 fallback 都正常。
- 200% 浏览器缩放下，建筑边缘、云层、水面细节应比 v2 更清楚。

## 工具备注

首选 Topaz Gigapixel 的 Art & CG 或 High Fidelity 2x 输出；备选 Upscayl 本地批量处理。Krea、Magnific、Freepik 一类创意增强图只作为候选，必须通过上面的坐标和结构检查后才能替换母图。
